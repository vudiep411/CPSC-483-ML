# CPSC-483-ML
## Requirements
Python 3.10

Install pipenv
```
$ pip install pipenv

```
Specify Python version
```
$ pipenv --python 3.10

```

Install all dependencies
```
$ pipenv install

```
Run the app

```
$ pipenv run streamlit run app.py
```


Activate the virtual shell env
```
$ pipenv shell
$ python --version

```
