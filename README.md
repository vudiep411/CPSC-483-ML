## Requirements for UI
Install Python 3.10

Install pipenv
```
$ pip install pipenv

```
Go to streamlit_ui dir
```
$ cd streamit_ui

```


Specify Python version
```
$ pipenv --python 3.10

```

Install all dependencies
```
$ pipenv install

```
Run the app

```
$ pipenv run streamlit run app.py
```


Activate the virtual shell env if you want to run notebook
```
$ pipenv shell
$ python --version

```
